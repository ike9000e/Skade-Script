static

Calculates line and column number given position in the original string.

Returned line number and column number are both 0-based.

Parameters
szOriginal- original c-string the position 'pos' was from.
length2- length of 'szOriginal', if -1 then null terminated.
pos- 0-based position.
szPosAlt- if 'pos' set to -1, alternate position as a pointer inside 'szOriginal'.
col- optional, output. column number.
ouLine- optional, output. the whole line extracted from the 'szOriginal'.
Returns
0-based line number, aka row number.
