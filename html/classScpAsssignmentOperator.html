Detailed Description

template<class TVar>
class ScpAsssignmentOperator< TVar >

Assignment operator, the '='.

Note that there is a difference in comparision versus plus or multiplicative operators. Instead of creating a new object instance, one of the operand objects, normally left, must be modified and pointer to it returned.
