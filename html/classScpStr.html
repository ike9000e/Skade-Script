Detailed Description

String-argument-replacement class.

Uses 'a' text-tag entries as a replacement in the input format string. Simple text str-replace matching, escape characters not supported. Triggers runtime assertions when detects wrong number of tag occurences compared to the number of arguments inserted with the .a() method(s).

// Example
str += *ScpStr("The item count is %a and color is %a.")
.a( nCount );
.a( bBlack ? "black" : "white" )
