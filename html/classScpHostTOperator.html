Detailed Description

template<class TVar>
class ScpHostTOperator< TVar >

Generic Template class for implementing operators like assignment or addition.

For importing objects from the host environment (C++ end) into the scripts. Requirements on a class that is passed as templeate parameter 'TVar':

  • must be derived from ScpHostObject.
  • must provide appropriate, one or more, C++ operator functions, that, in turn, classes derived from this (ScpHostTOperator) use.

There already exist, less generic, handfull list of template classes for operators: "*", "+" and "=", respectivelly: ScpPlusOperator, ScpMultiplicativeOperator or ScpAsssignmentOperator. If they can't provide sufficient features, this class can be used to reimplement them.
