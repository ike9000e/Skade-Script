Flags for ScpScript::eval3().

Enumerator
SCP_AF_AutoParse 

Can be set at the evaluation step to perform required parsing automatically.

Normally, user calls ScpScript::parse2() then ScpScript::eval2(), both functions manually. Using this flag enables parsing and evaluating to be done in one step, durning ScpScript::eval2() call.
